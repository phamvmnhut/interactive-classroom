## Quick start

To run website locally, you'll need to:

1. Clone the demo project from GitHub:

```
git clone https://github.com/phamvmnhut/interactive-classroom
```

2. Install packages required for work:

```bash
npm install
```

3. Run demo site:

```bash
npm start
```
